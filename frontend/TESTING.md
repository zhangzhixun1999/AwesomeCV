## 🚀 快速测试流程

1. **打开登录页**
   ```
   http://localhost:5173/login
   ```

2. **点击登录**（使用预填充的测试账号）

3. **查看仪表盘**
   - 应该看到 3 个简历卡片

4. **点击"创建新简历"**
   - 跳转到模板选择页

5. **选择一个模板**
   - 跳转到编辑器

6. **编辑简历内容**
   - 修改个人信息
   - 添加工作经历
   - 观察自动保存状态

7. **查看右侧预览**
   - 确认内容实时更新

8. **返回仪表盘**
   - 点击左上角"简历生成器"

9. **删除一个简历**
   - 点击删除按钮测试
