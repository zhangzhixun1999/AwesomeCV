# 🧪 前端功能快速测试

## 🚀 快速开始

### 第一步：登录

1. 打开浏览器访问：**http://localhost:5173/login**

2. 你会看到登录页面，已经预填充了测试账号：
   - 邮箱：`test@example.com`
   - 密码：`password123`

3. 点击蓝色的"登录"按钮

4. 等待约 0.8 秒（模拟网络请求），会自动跳转到仪表盘

### 第二步：查看仪表盘

登录成功后，你会看到：

**仪表盘页面** - 显示 3 个简历卡片：

   - 标题：软件工程师简历

   - 操作：编辑、复制、下载、删除

2. **产品经理 - 互联网方向** (商务风格 - 粉红渐变)
   - 标题：产品经理 - 互联网方向
   - 更新时间：2024年1月10日

3. **UI/UX 设计师作品集** (创意风格 - 蓝色渐变)
   - 标题：UI/UX 设计师作品集
   - 更新时间：2024年1月5日

4. **创建新简历卡片**（虚线边框）

---

### 第三步：测试模板选择

1. 点击"创建新简历"卡片（或点击右上角的"创建新简历"按钮）

2. 进入模板选择页面，看到 3 个模板：
   - **现代简约**（热门标签）
   - **商务专业**
   - **创意设计**（新标签）

3. 每个模板显示：
   - 渐变色预览图
   - 模板名称
   - 模板描述
   - 适用岗位标签
   - "使用此模板"按钮

---

### 第四步：测试编辑器

1. 在仪表盘中点击任意简历的"编辑"按钮

2. 进入编辑器页面，看到：
   - 左侧：编辑区
   - 右侧：实时预览区

3. **测试个人信息编辑**：
   - 修改姓名、邮箱、电话等
   - 右侧预览实时更新
   - 观察右上角保存状态（未保存 → 保存中... → 已保存）

4. **测试工作经历**：
   - 点击"添加工作经历"
   - 填写信息
   - 删除工作经历

5. **测试自动保存**：
   - 修改任意字段
   - 等待 2 秒
   - 看到状态变为"已保存"

2. 点击"删除"按钮

3. 查看右侧预览，姓名应该立即更新
4. 修改职位为"测试职位"
5. 预览区应该显示新职位

1. 打开编辑器
2. 修改个人信息
3. 观察右上角：

   - 修改后显示：`未保存`
   - 2秒后显示：`保存中...`
   - 完成后显示：`已保存`

## ⚠️ 注意事项

1. **不要刷新编辑器页面** - 会导致未保存的更改丢失
2. **使用测试账号登录** - 其他账号会提示错误
3. **等待自动保存** - 修改后等待2秒让自动保存完成
4. **查看浏览器控制台** - 如有问题，按 F12 查看错误

## 🔍 调试技巧

### 查看 localStorage
```javascript
// 在浏览器控制台执行
console.log(JSON.parse(localStorage.getItem('user')))
```

### 清除登录状态
```javascript
localStorage.clear()
location.reload()
```

### 查看当前路由
```javascript
window.location.pathname
```
